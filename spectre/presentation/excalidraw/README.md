# Excalidraw

Hand-drawn style diagrams for slides: [excalidraw.com](https://excalidraw.com) (free, runs in the browser).

## From Mermaid

1. Open [Mermaid Live](https://mermaid.live), paste a diagram from `../mermaid/diagrams.md` or copy from `../html/atlas.html` (view source, inside `<pre class="mermaid">`).
2. Export SVG or PNG from Mermaid Live.
3. In Excalidraw: **Menu → Import** and drop the image, *or* use it as a reference and redraw with arrows you can nudge by hand.

Some Excalidraw builds also support Mermaid import from the command palette — check **Help** for “Mermaid” if your version has it.

## Starter file

`pipeline-starter.excalidraw` is a minimal scene (a few labeled boxes). Open it in Excalidraw and extend it; it is **not** auto-synced with code.

## Tips for thesis slides

- Use **Frame** tool to size 16:9 regions.
- **Export** → PNG (2×) or SVG for LaTeX Beamer / PowerPoint.
