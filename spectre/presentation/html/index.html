Code & data

The pieces that produced everything above.

  • spectre/src/main.cpp- custom libllama speculative decoder with structured-output instrumentation
  • results/spectre/<run-id>/- per-run meta.json + tokens.csv; one directory per spectre invocation
  • spectre/scripts/quality_eval.py- builds the 7 quality figures from results/spectre/*/
  • spectre/scripts/benchmark.sh- sweeps the spectre binary over (seed × n_max); writes results/spectre/<run-id>/
  • spectre/scripts/README.md- pipeline overview + model-pair guidance
  • llama_cpp_benchmark.sh- llama-server benchmark across all --spec-type values (separate engine; logs in logs/)
  • llama_cpp_benchmark_speculative.sh- llama-server speculative variants of the above
