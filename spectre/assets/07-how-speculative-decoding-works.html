Sequence — one speculative round (conceptual)

  1. Autoregressive baseline: one target forward per emitted token — simple, slow.
  2. Idea: cheap draft runs ahead; expensive target checks many positions at once.
  3. Correctness: accepted tokens are exactly as if sampled from the target; rejections trigger a proper resample (Leviathan / speculative sampling).
  4. Speedup: when drafts align with the target, you emit several tokens per target pass.
